# Challenge Summary
- Writing a function called `validate brackets` to check 3 types of brackets if the opening and closing bracket aligns correctly:
- Round Brackets : `()`
- Square Brackets : `[]`
- Curly Brackets : `{}`

- > Arguments: string
- > Return: boolean
